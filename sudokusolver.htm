How it works

This program should solve all "solvable" Sudoku puzzles, by which I mean those that do not require guesswork to complete but instead rely only on logic.

It uses five solving functions to work its way through a puzzle. The functions eliminate possible values in each square until there is only one left. The solving functions are:

  1. Does any square have only 1 possible value remaining?
    If so, that square must be that value.
  2. Does any square contain the last instance of a value in it's row, column or supercell (a supercell is 1 of the 9 larger 3 x 3 squares that makes up the overall 9 x 9 grid)?
    If so, that square must be that value.
  3. Do x values only occur together x times in a row, column or supercell ?
    If so, then those x values cannot occur outside of those cells in the related row, column or supercell squares.
  4. Do any unsolved squares in a supercell contain a value in the same row or column?
    If so, that row or column can be extended into other unsolved supercells and the value removed from them.
  5. Within two of the three cells in a supercell row or column, do the patterns of values occurring in those two supercells' sub-rows or sub-columns, limit where that value can occur in the third supercell?
    If so, remove that value from the appropriate sub-row or column in the third supercell.
