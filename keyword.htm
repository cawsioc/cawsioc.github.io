How it works

The Keyword Substitution Cipher uses the letters from a keyword or phrase as the start of a cipher-text alphabet, and then appends any characters that didn't occur in the keyword on to the end of that alphabet to give the full character set required for substitution. To encrypt, the position of the plain-text character in the plain-text alphabet is used to get the corresponding cipher-text character from the ciphe-rtext alphabet. To decrypt, the receiver constructs the cipher-text alphabet as above and finds the the plain-text character with position corresponding to that of the cipher-text character in the cipher-text alphabet. (403,291,461,126,605,635,584,000,000) different possibilities, but using real words or phrases that contain only a subset of the full alphabet's letters as the key removes a huge number of these as candidate cipher-text alphabets. Frequency analysis is a useful tool when attempting to break this cipher.
